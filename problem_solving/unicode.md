## 알파벳 대, 소문자 유니코드로 활용하기

```
string.charCodeAt()             // 해당하는 알파벳의 유니코드 값 리턴, string은 알파벳을 할당한 변수

65 <= 대문자 아스키 코드 값 <= 90    // 알파벳 대문자는 65 이상 90 이하의 값을 가짐

97 <= 소문자 아스키 코드 값 <= 122   // 알파벳 소문자는 92 이상 122 이하의 값을 가짐

공백 === 32                      // 공백의 경우 32 값을 가짐

String.fromCharCode(number)     // 매개 변수 number에 해당하는 유니코드 문자를 리턴


```
